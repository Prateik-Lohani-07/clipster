# Clipster

Clipster is an easy-to-use, lightweight clipboard, provided as a chrome extension.

It has a simple 90's websites inspired UI.
Built with: React + Typescript + Tailwind.

<img src="clipster.png" alt="Snapshot of Clipster as an extension" width="300" />

Clipster stores text right in your browser, in your local storage. You can click on the text box as many times as you want to copy the text.

Additionally, aside from the default color, Clipster also provides color coding with upto 6 different colors so you can copy text faster visually.
